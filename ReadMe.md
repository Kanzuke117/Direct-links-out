# ![logo](https://raw.githubusercontent.com/XX-J/Direct-links-out/master/icon.png) Direct links out
Userscript that removes all this "You are leaving our site..." and other redirection stuff from links.

- Better anonymity
- Faster surfing

- 4PDA
- Addons.Mozilla.Org
- DanielDefo
- DeviantArt
- Disq.us
- ForumAvia
- Facebook
- Google
